A Tab is a child of the Tabs component and is used to toggle the visibility of a TabPanel.

### Inactive Tab

```jsx
import { useState } from 'react';
import { Tab } from '@atomikui/core';

<Tab label="Tab One" />;
```

### Active Tab

```jsx
import { useState } from 'react';
import { Tab } from '@atomikui/core';

<Tab label="Tab One" active />;
```
